Control Room (during telescope/trailer shutdown)
  • On control04 choose Select Flasher Run from VAC
  • On control04 choose End Night from VAC and exit VAC
  • On harvester, stop the errorArchivingDemon: ssh harvester ./errorArchivingDemon_kill.sh
  • On control02 ssh harvester and run nightsum and put the results in the ELog
  • On control02 copy/paste the text runlog from the Run Log Generator into the ELog
    (Either select the text format on the website or click the LG icon on control02, which should work even if romulus is down)
  • On control02 send the ELog
  • On control02 reset CFD thresholds if necessary (depending on moonlight)
  • On control01 close the L2 trigger GUI
Control Room (after telescope/trailer shutdown)
  • On control04 verify that all four telescope interlocks are on and then exit the tracking programme
  • On control02 verify that all four shutters are closed in the Pointing Monitor focal plane view
  • On control02 power off T1-T4 Pointing Monitors using Power Off All
  • On control02 after all Pointing Monitors have powered off, close the Pointing Monitor Control window
  • On control01 verify that all three FIRs are powered down (windows red) and close the FIR windows
  • On control01 verify that the Allsky monitor lid is on and close the Allsky Monitor window
  • Update the telescope operation time/moon status on the whiteboard in the control room.
  • Close ALL terminals and windows on the control machines. Log off from the account.
